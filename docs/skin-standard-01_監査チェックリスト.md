# skin-standard-01 監査チェックリスト v1.0

監査の問い：**「¥165,000の仕事に見えるか。安いテンプレに見える瞬間が1箇所でもないか」**

グリッド：3ペア（classic / modern / soft）× 色2系統（暗色main / 明色main）＝ **6パターン**
確認ページ：TOP・子猫詳細・下層ページ（法定表示）
確認幅：PC（1280以上）と SP（375）

---

## 準備：色2系統の設定ブロック値

### 系統1：暗色main（現在の既定値。差し替え不要）

```scss
--skin-base:   #F6F4EF;
--skin-main:   #46586A;
--skin-accent: #937218;
```

### 系統2：明色main（監査用テストパレット。数値検証済み・全項目4.5:1以上）

```scss
--skin-base:   #FAF7F2;
--skin-main:   #D9B8A8;  /* 淡いロゼ。明度上限0.52で実効 #7F6253 に沈む */
--skin-accent: #8A9B8E;  /* セージグリーン */
```

**手順**：系統1で3ペアを監査 → 設定ブロックを系統2に差し替え→ビルド→3ペアを監査 → **必ず系統1に戻してビルド**（戻し忘れ防止：最後に `git diff` で設定ブロックが既定値なことを確認）

## 各グリッドのチェック項目（6パターン共通）

### 色の可読性（明色main系統で特に注視）
- [ ] ボタン文字（ネイビー実ボタンの白文字）が読める
- [ ] 見出しラベル（GREETING等の英字）が背景から浮かず読める
- [ ] バッジ3種（募集中／商談中／ご家族決定）の白文字が読める
- [ ] 金の罫（ヒーロー・セクション見出し）が背景に消えていない

### ペアの世界観
- [ ] classic：明朝・カード1px線・かっちりした角。**影が出ていないこと**
- [ ] modern：M PLUS 1p・影・完全な直角。**線が出ていないこと**
- [ ] soft：丸ゴ・影・大きな角丸・ピルボタン
- [ ] ナビの書体がペアに連動している
- [ ] CTAタイトルの書体・ウェイトがペアに連動している

### 触感・挙動
- [ ] ボタン3種のhover：色がふわっと沈む／白ボタンは淡青に（動かないこと）
- [ ] カードhover：0.2sでふわっと-4px＋影
- [ ] 商談中バッジの色がアクセント由来で違和感がない

### タイポ・折返し（SP 375で）
- [ ] ヒーロータイトル：改行バランスが自然（balance）
- [ ] CTA：孤立改行なし・文節で折れている（Chrome）／Safariでも文節折返し（BudouX）
- [ ] 長い英字の猫名（Chocolatepoint...）でカードが崩れない

### レイアウト
- [ ] ヒーロー3バリアント（split / photo / minimal）で崩れなし。photoでは罫が白
- [ ] フッター5段（ブランド／ナビ／法的リンク／取扱業／コピー）のまとまり
- [ ] ご家族決定の子猫詳細で**価格が非表示**（募集中・商談中では表示）

## グリッド外の最終確認

- [ ] **default無傷**：スキンをdefaultに戻し、素の見た目が一切変わっていない
- [ ] **yuki / nostalgia無傷**：切替して従来どおり（yukiはモーションrichで雪）
- [ ] **生HEXチェック**：`grep -rn "#[0-9a-fA-F]\{3\}" assets/scss --include="*.scss" | grep -v _tokens | grep -v skins/`（_mixins.scssのコメント1件のみ許容）
- [ ] **並置比較**：defaultとstandard-01をタブで並べ、「触れば分かる質感差」があるか
- [ ] **最終の問い**：6パターンを通しで眺め、「安いテンプレに見える瞬間」が1箇所でもないか。あれば要素名を控えて微調整へ
